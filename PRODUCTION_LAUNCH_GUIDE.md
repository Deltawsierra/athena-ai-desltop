## üîß Troubleshooting

### If the app doesn't start:
1. Make sure you ran `npm install` first
2. Verify the `dist` folder exists (created by `npm run build`)
3. Try `launch-production.bat` to see any error messages
4. Fall back to development mode with `launch-windows.bat`

### If you see a blank window:
1. Wait 5-10 seconds for the server to fully start
2. Press `Ctrl+R` to reload the window
3. Check if port 5000 is already in use

### To rebuild for updates:
1. Open command prompt in project folder
2. Run: `npm run build`
3. Launch with production launcher
